paperless-ngx-helm

A helm chart to install paperless-ngx.


The source of the Helm chart can be found here.


Using the Helm Chart

To use the paperless-ngx-helm chart, follow these steps:

  1. Add the helm chart repository:
    2. helm repo add paperless-ngx-helm https://wreiner.github.io/paperless-ngx-helm
  2. Update the repo index:
    3. helm repo update
  3. Install the chart:
    4. helm install my-paperless-ngx [-f values.yaml] [-n paperless-ns --create-namespace]
  4. Show all the versions available:
    5. helm search repo paperless-ngx-helm/paperless-ngx --versions
  5. Get the default values.yaml:
    6. helm show values paperless-ngx-helm/paperless-ngx > paperless-ngx-values.yaml
